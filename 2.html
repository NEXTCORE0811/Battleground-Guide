Battleground is famous game in the korea. purpose of this game is one-hundred people fight in arengel, miramar and sanook.

배틀그라운드는 한국에서 유명한 게임이다, 이 게임의 목표는 100명이 에렌겔, 미라마 그리고 사녹에서 서로 싸우는것이다.

it is survival game and strategy game. if you want to win the game you must be use your medical kits, (Bandage, energy drink, pain killer pills, first aid kit, med kit, Adrenalin Syringe)

이 게임은 전략적인 게임이자, 생존게임이다. 만약 당신이 이 게임에서 이기고싶다면 당신은 반드시 딩신의 아이템인 의료용품, (붕대, 에너지 드링크, 진통제, 구급상자, 의료용 키트, 아드레날린 주사기)

grenades, (freg grenade, stun grenade, smoke grenade, molotov cocktail) armors (1 level bulletproof hat, a flak suit, backpack / 2 level bulletproof hat, a flak suit, backpack / 3 level bulletproof hat, a flak suit, backpack)

수류탄들, (폭파 수류탄, 섬광탄, 연막탄, 화염병) 보호장비 (1레벨 방탄모, 방탄복, 가방 / 2레벨 방탄모, 방탄복, 가방 / 3레벨 방탄모, 방탄복, 가방)들을 써야 한다

and also There is a magnetic field inside the game. magnectic field is very fatal. so if you don't want to get hurt, you will think your own plan.

그리고 또한 이 게임 안에는 자기장이 있다. 자기장은 매우 치명적이다. 그래서 만약 당신이 다치기 싫다면, 당신은 자기만의 작전을 세워야만 할것이다.

it have four game mode (normal mode : the classic battle royale experience / zombie mode : humans vs zombies / war mode : death match with respawns / esport mode : battle royale with official 2018 esports rule settings)

이 게임은 4가지 종류의 게임 모드가 있다. (노멀 모드 : 배틀로얄의 고전적인 모드 / 좀비 모드 : 인간 대 좀비 / 워 모드 : 리스폰이 되는 데스메치 경기 / e스포츠 모드 / 2018 e스포츠 룰이 적용된 배틀로얄 게임 )
